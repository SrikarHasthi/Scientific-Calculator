# Scientific-Calculator
Scientific Calculator using python and GUI in python.
When you run the code the 1st frame will be the standard calculator.
If you want to switch to scientific calculator press the file button on top left corner and select scientific calculator.
To use the trigonometry functions first enter the value and then press the trigonometry functions.
****NOTE******
Entering values with keyword will not work.You have to press the button on the frame.
